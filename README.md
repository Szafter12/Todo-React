
# ToDo App react

It's a simple todo app made in react creacted for practise useState and tailwindCSS. 


## Demo

https://szafter12todoreact.netlify.app/


## Run Locally

Clone the project

```bash
  git clone https://github.com/Szafter12/Todo-React
```

Go to the project directory

```bash
  cd Todo-React
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm run dev
```


## Features

- Adding tasks
- Task counter
- Completion of tasks
- Deleting tasks
- Responsive desing


## Tech Stack

- React 
- TailwindCSS
- Vite
